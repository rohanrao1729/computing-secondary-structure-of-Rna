DAA Assignment 2
Problem Statement

Given an RNA structure as a string, we need to find the optimal number of pairings in the secondary structure such that
A pairs with U
C pairs with G
and each pair is atleast at a distance greater than or equal to 4
and there are no intersecting pairs

Input format

Enter the number of test cases
Enter the the input string for each testcase.
tc
string1
string2
.
.
stringtc

Output

This program will report the maximal number of pairings in the secondary structure of the given RNA string .
